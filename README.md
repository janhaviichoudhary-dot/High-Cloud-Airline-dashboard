# Airline-power-bi-dashboard-
This project aims to gain a comprehensive understanding of "High Cloud Airlines" operations through data analysis and extract valuable insights to optimize high cloud airline operations and performance. This will involve investigating load factors, identifying top carrier names based on passenger preference, analyzing popular routes, and exploring other key metrics. The ultimate goal is to provide actionable recommendations that can enhance operational efficiency and profitability.
